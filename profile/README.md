# 라플라플 (LaPlaPl)

**가계부 기반 맞춤형 공공정책 추천 서비스**

---

## 프로젝트 소개

라플라플은 청년들이 정부 지원 혜택을 놓치지 않도록 돕는 서비스입니다.

가계부를 통해 사용자의 재정 상황을 파악하고, 생애주기와 상황에 맞는 공공정책을 자동으로 추천합니다.

## 해결하고자 하는 문제

- **정보 분산** - 여러 부처와 지자체에 흩어진 정책 정보
- **복잡한 자격 요건** - 본인 해당 여부 파악의 어려움
- **신청 시기 누락** - 놓치는 혜택들
- **재정 관리 부재** - 적절한 지원을 찾지 못하는 상황

## 주요 기능

| 기능 | 설명 |
|------|------|
| 스마트 가계부 | 수입/지출 기록, 예산 설정, 소비 패턴 분석 |
| 맞춤형 정책 추천 | 사용자 프로필과 재정 데이터 기반 정책 매칭 |
| 대출 정보 제공 | 청년 대상 정부 대출 및 금융권 대출 비교 |
| 알림 서비스 | 신청 기간, 신규 정책, 자격 충족 알림 |

## 타겟 사용자

대학생 · 취업준비생 · 사회초년생 · 예비부부 · 신혼부부 · 1인 가구

## 기술 스택

- **Frontend**: React Native (iOS / Android)
- **Backend**: Node.js / Python
- **Database**: PostgreSQL + Redis
- **Infra**: AWS / GCP
